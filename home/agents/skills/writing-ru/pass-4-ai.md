# Проход 4: AI-паттерны

У этих паттернов одна причина: модель тянет к среднему и сглаживает конкретику в общие, важно звучащие фразы. Одна проверка ловит большинство: подходит ли предложение почти к любому предмету? Если да — вырежи его или замени тем, что верно только для этого предмета.

## Псевдокод

```
for each предложение:
  if начинается с маркера («Важно отметить», «В современном мире», «Давайте разберёмся»):
    убери маркер, начни с сути  // Убирай фразы-маркеры AI-текста

for each прилагательное/модификатор:
  if рекламный штамп (широкий ассортимент, индивидуальный подход, инновационный):
    замени фактом, числом или названием  // Заменяй рекламные штампы конкретикой

for текст в целом:
  if все предложения одинаковой длины (однообразные 15–20 слов):
    чередуй: добавь короткие, дай некоторым вытянуться  // Чередуй короткие и длинные
  if связки избыточны («Кроме того», «Также», «В заключение»):
    убери, если логика и так понятна  // Ломай однообразную AI-структуру

for each пунктуация:
  if точка с запятой и хотя бы одно предложение длинное:
    разбей на два предложения  // Упрощай пунктуацию
  if скобки с важной информацией:
    вынеси в отдельное предложение  // Упрощай пунктуацию
  if больше трёх запятых:
    разбей на два предложения  // Упрощай пунктуацию

for each предложение:
  if раздувает значимость/наследие/тренд или кончается деепричастным оборотом-оценкой:
    вырежи или замени конкретным фактом с источником  // Убирай раздувание значимости
  if связку заменили на «является»/«представляет собой»/«выступает в качестве»:
    верни тире или прямой глагол  // Возвращай тире вместо «является»
  if «не просто X, а Y» / «это не X, это Y» / тройка-перечисление:
    скажи прямо; оставь в тройке только то, что верно  // Убирай шаблонные конструкции

for each слово:
  if из набора AI-лексики и он скапливается здесь:
    возьми простой глагол  // Прорежай AI-лексику
  if размытая атрибуция («эксперты считают», «ряд источников»):
    назови источник и точное утверждение или убери  // Прорежай AI-лексику
  if синоним подставлен только чтобы не повторяться:
    повтори простое слово  // Прорежай AI-лексику

guard:
  оставь простые формы, простые слова и настоящие превосходные степени; одиночный случай — не признак  // Сохраняй живое и понятное
```

## Правила

### Убирай фразы-маркеры AI-текста

Фразы-маркеры выдают машинный текст. Убирай их и начинай с сути.

| Было                                               | Стало                             |
| -------------------------------------------------- | --------------------------------- |
| Важно отметить, что система работает стабильно     | Система работает стабильно        |
| В современном мире технологии играют ключевую роль | Технологии влияют на X и Y        |
| Давайте разберёмся, как это работает               | Вот как это работает              |
| Таким образом, можно сделать вывод                 | (убрать, написать вывод напрямую) |

**Частые маркеры:** «важно отметить», «в современном мире», «на сегодняшний день», «является одним из», «не секрет, что», «играет ключевую роль», «следует учитывать».

### Заменяй рекламные штампы конкретикой

Рекламные штампы не несут информации. Замени каждый штамп фактом, числом или названием.

| Было                     | Стало                                                        |
| ------------------------ | ------------------------------------------------------------ |
| широкий ассортимент      | 200 товаров в 5 категориях                                   |
| индивидуальный подход    | назначаем менеджера, который ведёт проект от начала до конца |
| инновационные технологии | используем технологию X                                      |
| конкурентные цены        | от 500 ₽, дешевле рынка на 15%                               |

**10 частых штампов:** широкий ассортимент, индивидуальный подход, высокое качество, минимальный уход, конкурентные цены, инновационные технологии, быстрая доставка, широкий спектр, ряд преимуществ, идеальное решение.

### Ломай однообразную AI-структуру

AI-текст предсказуем: одинаковая длина предложений, каждый абзац с обобщения, избыточная «сбалансированность». Ломай шаблон.

| Было                                                                 | Стало                                                                      |
| -------------------------------------------------------------------- | -------------------------------------------------------------------------- |
| Каждый абзац начинается с обобщения, затем 3 пункта одинаковой длины | Начни один абзац с примера, другой — с вопроса, третий — с короткого факта |
| С одной стороны X, с другой стороны Y (когда позиция однозначна)     | X. Точка.                                                                  |
| Также следует отметить... Кроме того... В заключение...              | Убрать связки, если абзацы и так логически связаны                         |

**Приёмы:** чередуй длину предложений, вставь конкретный пример, убери «сбалансированные» оговорки, если позиция однозначна.

### Чередуй короткие и длинные предложения

Текст из одинаковых предложений усыпляет. Чередуй короткие и длинные — ритм удерживает внимание.

| Было                                                                                                           | Стало                                                                                     |
| -------------------------------------------------------------------------------------------------------------- | ----------------------------------------------------------------------------------------- |
| Система обрабатывает запросы. Система проверяет данные. Система возвращает результат. Система логирует ошибки. | Система обрабатывает запросы и проверяет данные. Ошибки логируются. Результат — за 50 мс. |

**Приём:** после длинного объяснения — короткое предложение. Это работает.

### Упрощай пунктуацию

Точка — лучший знак препинания. Если сомневаешься в пунктуации — упрости предложение.

| Было                                                                                                                | Стало                                                                                   |
| ------------------------------------------------------------------------------------------------------------------- | --------------------------------------------------------------------------------------- |
| Система, которая была разработана нашей командой, и которая, по мнению экспертов, является лучшей, запущена в прод. | Наша команда разработала систему. Эксперты считают её лучшей на рынке. Система в проде. |
| Результат (впрочем, не окончательный) оказался неожиданным.                                                         | Предварительный результат оказался неожиданным.                                         |
| Мы хотели бы отметить... что проект движется вперёд.                                                                | Проект движется вперёд.                                                                 |

**Заменяй тире другой пунктуацией при склейке предложений.** AI-текст злоупотребляет тире как универсальным соединителем. Вместо тире: раздели на два предложения, используй союз, двоеточие или запятую. Тире допустимо только в устойчивых конструкциях («один абзац — одна мысль») и при обособлении приложений.

**Дефис (-) и тире (—):** дефис только внутри слов (какой-то, онлайн-курс), без пробелов. Тире длинное, с пробелами. В русской типографике среднего тире нет.

**Больше трёх запятых** в предложении: разбей на два.

**Точка с запятой** допустима, когда оба предложения короткие (см. проход 2). Если хотя бы одно длинное — разбей точкой.

**Скобки:** информация в них выглядит второстепенной. Убери или вынеси в отдельное предложение. Оставляй только для ссылок и источников.

**Многоточие:** только для пропусков в цитатах. Для неуверенности и пауз не годится.

### Убирай раздувание значимости

Фразы про важность, наследие или «более широкие процессы» ничего не говорят о предмете. Убери их или замени конкретным фактом с источником. Деепричастный оборот-оценка в конце предложения — то же самое на уровне оборота.

| Было                                                                                            | Стало                         |
| ----------------------------------------------------------------------------------------------- | ----------------------------- |
| Институт основан в 1989 году, что стало поворотным моментом в развитии региональной статистики. | Институт основан в 1989 году. |
| На станции восемь путей, что вносит вклад в социально-экономическое развитие региона.           | На станции восемь путей.      |
| Эта этимология подчёркивает непреходящее наследие сопротивления общины.                         | (убрать)                      |

**Маркеры:** играет ключевую роль, знаменует поворотный момент, оставил неизгладимый след, отражает более широкие процессы, подчёркивает значимость, открывает новую главу, в эпоху.

### Возвращай тире вместо «является»

В русском настоящем времени связки нет: между подлежащим и сказуемым ставят тире. AI подставляет «является», «представляет собой», «выступает в качестве». Верни тире или прямой глагол.

| Было                                                     | Стало                                          |
| -------------------------------------------------------- | ---------------------------------------------- |
| Галерея является выставочным пространством ассоциации.   | Галерея — выставочное пространство ассоциации. |
| Платформа представляет собой одно из крупнейших решений. | Платформа — одно из крупнейших решений.        |
| Компания выступает в качестве поставщика оборудования.   | Компания поставляет оборудование.              |

**Замены:** является / представляет собой / выступает в качестве / служит → тире или прямой глагол.

### Убирай шаблонные конструкции

Две привычные машинные конструкции. «Не просто X, а Y», «это не X, это Y» — выдуманное возражение, которое тут же опровергают. Тройка-перечисление добавляет к одному смыслу два почти-синонима. Скажи прямо; оставь только то, что верно.

| Было                                                         | Стало                                 |
| ------------------------------------------------------------ | ------------------------------------- |
| Портрет — не просто автопортрет, а документ её одержимостей. | Портрет документирует её одержимости. |
| Ясное, краткое и убедительное резюме.                        | Ясное резюме.                         |
| Платформа быстрая, надёжная и масштабируемая.                | Платформа держит 50 000 соединений.   |

### Прорежай AI-лексику

Набор повторяющихся слов выдаёт машинный текст плотностью, а не одним употреблением. Где набор скапливается, бери простой глагол. Рядом идут два спутника.

**Набор:** подчёркивает, отражает, играет ключевую роль, в условиях, на фоне, ландшафт, палитра, неотъемлемая часть, богатое наследие, динамично развивающийся, призван, способствует, целый ряд, уникальный, комплексный.

- **Размытая атрибуция:** «эксперты считают», «по мнению наблюдателей», «ряд источников» (когда источник один). Назови источник и точное утверждение или убери.
- **Подмена синонимов:** «компания… фирма… организация». Повтори простое слово («компания… она… компания»). Перебор синонимов — машинный приём, человек повторяет.

### Сохраняй живое и понятное

Признак — плотность и совпадение, а не один случай. Оставь живое и понятное:

- Простые формы: «это», прямую форму без глагола, тире.
- Простые слова вместо вычурных: написал (не «осуществил написание»), использовал (не «задействовал»), умер (не «ушёл из жизни»).
- Настоящие превосходные степени, когда это факт: первый, единственный, крупнейший.
- Одно тире, одно «Кроме того» — не признак AI. Останови правку, пока текст остаётся живым.
