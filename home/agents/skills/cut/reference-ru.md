# Русские правки — справка для встроенного рубрикатора в cut.ts

Сжатый рубрикатор в `cut.ts` (RUBRIC_RU) выведен из этих четырёх проходов. Этот файл — полный источник; правь его, затем перегоняй в константу.

# Проход 1: Слова

## Псевдокод

```
for each слово/фраза in текст:
  if существительное несёт действие (отглагольное: -ание, -ение, -ция):
    замени глаголом  // Ставь глагол в центр предложения
  if есть более короткое слово с тем же смыслом:
    замени коротким  // Выбирай короткое слово
  if слово пустое (паразит, дублирование, очевидное):
    убери  // Убирай слова без смысловой нагрузки
  if отрицательная конструкция (отрицание или контраст «X, а не Y»):
    замени утвердительной  // Пиши утверждениями
  if мёртвая метафора (стёртый образ):
    замени конкретным утверждением  // Заменяй мёртвые метафоры конкретикой
  if канцелярит или книжное слово с живым аналогом:
    замени живым языком  // Пиши живым языком
```

## Правила

### Ставь глагол в центр предложения

Глагол — двигатель предложения. Отглагольные существительные (-ание, -ение, -ция) утяжеляют текст и прячут действие.

| Было                             | Стало                    |
| -------------------------------- | ------------------------ |
| осуществление поддержки клиентов | поддерживаем клиентов    |
| произвести установку             | установить               |
| выполнить оптимизацию            | оптимизировать           |
| обеспечить функционирование      | запустить / поддерживать |

**Тест:** если можно убрать «произвести/осуществить/выполнить» и поставить глагол — убирай.

### Выбирай короткое слово

Короткое слово понятнее длинного.

| Было                      | Стало    |
| ------------------------- | -------- |
| на сегодняшний день       | сейчас   |
| осуществлять деятельность | работать |
| произвести оплату         | оплатить |
| в целях обеспечения       | чтобы    |

### Убирай слова без смысловой нагрузки

Каждое слово должно нести смысл. Убрал слово, смысл остался — значит, слово было мусором.

| Было                                        | Стало               |
| ------------------------------------------- | ------------------- |
| Следует отметить, что система работает      | Система работает    |
| Ни для кого не секрет, что цены растут      | Цены растут         |
| В принципе, это работает достаточно хорошо  | Это работает хорошо |
| Как уже было сказано выше, данные обновлены | Данные обновлены    |

**Слова-паразиты:** «в принципе», «в целом», «как бы», «собственно говоря», «достаточно», «довольно», «весьма».

**Тест:** убери слово. Смысл сохранился? Значит, слово было мусором.

### Пиши утверждениями

Отрицания требуют дополнительного усилия: читатель сначала представляет действие, потом отменяет его. Утверждение сразу даёт правильную картину.

| Было                    | Стало          |
| ----------------------- | -------------- |
| Не забудьте сохранить   | Сохраните      |
| Он не помнил            | Он забыл       |
| Нельзя не отметить      | Стоит отметить |
| Не является невозможным | Возможно       |

**Двойное отрицание снижает ясность.** «Не следует игнорировать» → «Учитывайте».

**Контрастное отрицание** («X, а не Y» и «Не X, а Y»): перепиши без «не». Если «не X» несёт факт — замени «а не» на «вместо» / «только». Если «не X» — мёртвая метафора или пустой контраст — убери «не X» целиком.

| Было                          | Стало                          |
| ----------------------------- | ------------------------------ |
| Улучшился X, а не Y           | Улучшился только X             |
| Мы выбрали X, а не Y          | Мы выбрали X вместо Y          |
| Используй X, а не Y           | Используй X вместо Y           |
| X изменился, а не Y           | Y остался прежним; X изменился |
| Не марафон, а устойчивый ритм | Устойчивый ритм                |
| Не теория, а практика         | Практика                       |

### Заменяй мёртвые метафоры конкретикой

Мёртвые метафоры — стёртые образы, которые когда-то создавали картинку, а теперь проходят незамеченными. Замени конкретным фактом или числом. Придерживайся буквальных формулировок.

| Было                               | Стало                      |
| ---------------------------------- | -------------------------- |
| вывести на новый уровень           | увеличить конверсию на 10% |
| ключевой фактор успеха             | главная причина роста      |
| двигаться в правильном направлении | сокращать время ответа     |
| заложить фундамент                 | написать первую версию API |
| открыть новые горизонты            | выйти на рынок Европы      |

**Тест:** если убрать метафору и написать, что именно произошло, текст станет конкретнее — значит, метафора была мёртвой.

### Пиши живым языком

Канцелярский язык отталкивает читателя. Пиши так, как говорят живые люди — но в пределах литературной нормы.

| Было                     | Стало             |
| ------------------------ | ----------------- |
| в рамках данного проекта | в этом проекте    |
| ввиду отсутствия         | так как нет       |
| имеет место быть         | есть / происходит |
| вышеуказанный            | этот              |
| в связи с тем, что       | так как           |
| при условии, что         | если              |

# Проход 2: Предложения

## Псевдокод

```
for each предложение:
  if глагол в пассивном залоге и деятель известен:
    перепиши в активном залоге  // Используй активный залог
  if причастный/деепричастный оборот длиннее 4 слов:
    разверни в придаточное с «который» или отдельное предложение  // Разворачивай причастные обороты
  if нет сказуемого и смысл держится на контрасте («но», «а не», «зато»):
    перепиши с подлежащим и сказуемым  // Разворачивай безглагольные обрывки

// Связность между предложениями (known-new contract, Williams)
for each абзац:
  for each соседняя пара (предложение N, предложение N+1):
    тема_N1 = подлежащее/тема предложения N+1
    конец_N = новая информация в конце предложения N
    if тема_N1 не подхватывает конец_N и не повторяет тему_N:
      перепиши: вынеси известное в начало N+1, новое в конец  // Известное → начало, новое → конец
  темы = [подлежащее каждого предложения в абзаце]
  if темы скачут (каждое предложение начинается с нового субъекта):
    перепиши с единой темой или склей предложения  // Единая тема в абзаце

// Цикл разбивки и склейки
loop:
  for each предложение:
    while больше одной мысли:
      разбей → цель 15–25 слов на кусок  // Разбей и склей
  for each соседняя пара:
    if об одном, но звучат как чужие:
      склей: союз / общее подлежащее / отсылка назад / слияние в оборот (причастия до 4 слов) / двоеточие
      if склеенное перегружено:
        перефразируй другой техникой
        if после 2 попыток всё ещё перегружено:
          оставь более простой из двух вариантов и продолжай
      go to loop

// Финальная проверка
for each предложение:
  if запинаешься при чтении вслух:
    перепиши  // Прочитай вслух
  if длина > 20 слов и сказуемое во второй половине:
    проверь: вступление необходимо или можно перестроить?  // Ставь сказуемое ближе к началу
```

## Правила

### Известное в начало, новое в конец

Читатель легко следит за мыслью, когда каждое предложение начинается с того, что он уже знает, а заканчивается новой информацией. Новое из конца предложения N становится известным в начале предложения N+1 (known-new contract, Williams).

| Было                                                                               | Стало                                                                                                 |
| ---------------------------------------------------------------------------------- | ----------------------------------------------------------------------------------------------------- |
| Полураспад 5–7 часов. Кофеин за 6 часов до сна снижает сон на 45 мин.              | Полураспад 5–7 часов, поэтому кофеин за 6 часов до сна сокращает сон на 45 мин.                       |
| Чтобы заснуть, тело сбрасывает 1–3°F. Тёплые конечности ускоряют теплоотдачу.      | Чтобы заснуть, тело сбрасывает 1–3°F, а тёплые конечности ускоряют этот процесс.                      |
| Верхний свет хуже всего: меланопсин в нижней сетчатке. Используй настольные лампы. | Верхний свет хуже всего, потому что меланопсин в нижней сетчатке, поэтому используй настольные лампы. |

**Диагностика:** подчеркни начало (тему) и конец (новое) каждого предложения. Связность работает, когда начало N+1 подхватывает конец N.

### Единая тема в абзаце

Предложения одного абзаца начинаются с одной темы или её вариации. Если каждое предложение начинается с нового субъекта, читатель теряет нить.

| Было                                                                                          | Стало                                                                                                           |
| --------------------------------------------------------------------------------------------- | --------------------------------------------------------------------------------------------------------------- |
| Кровать только для сна. Если не засыпаешь за 15–20 мин, встаёшь. Возвращаешься, когда сонный. | Кровать только для сна. Если не засыпаешь за 15–20 мин, уходишь в другую комнату и возвращаешься, когда сонный. |

**Диагностика:** выпиши подлежащее каждого предложения. Если все разные (A, B, C, D), перепиши с единым подлежащим или склей.

Источник: Williams, Style: Lessons in Clarity and Grace, Lesson 5 "Cohesion and Coherence", принципы 4, 6, 7.

### Используй активный залог

Активный залог показывает, кто действует. Пассивный прячет деятеля и усложняет понимание.

| Было                           | Стало                      |
| ------------------------------ | -------------------------- |
| Запрос обрабатывается сервером | Сервер обрабатывает запрос |
| Решение было принято           | Команда приняла решение    |
| Ошибка была обнаружена         | Тест обнаружил ошибку      |
| Данные были загружены          | Система загрузила данные   |

**Исключение:** пассив допустим, когда деятель неизвестен или неважен: «Письмо отправлено.»

### Разворачивай причастные обороты

Причастные и деепричастные обороты перегружают предложение. Разверни в придаточное с «который» или отдельное предложение.

| Было                                               | Стало                                                   |
| -------------------------------------------------- | ------------------------------------------------------- |
| Пользователь, воспользовавшийся сервисом           | Пользователь, который воспользовался сервисом           |
| Он был прав, спрашивая вас                         | Он правильно сделал, что спросил вас                    |
| Обеспечивая надёжность, система работает стабильно | Система работает стабильно: она обеспечивает надёжность |

**Тест:** если причастный оборот длиннее четырёх слов — разворачивай.

### Разворачивай безглагольные обрывки

Фрагменты без сказуемого, которые держатся на контрасте («но», «а не», «зато»). Читатель сам достраивает пропущенный глагол. Скажи прямо, что произошло.

| Было                                              | Стало                                |
| ------------------------------------------------- | ------------------------------------ |
| Реальный прогресс, но в окружении, а не в модели. | Улучшилось только окружение.         |
| Хорошие результаты, но только локально.           | Результаты улучшились локально.      |
| Больше фич, меньше стабильности.                  | Добавление фич снизило стабильность. |
| Быстро, но хрупко.                                | Работает быстро, но легко ломается.  |

**Тест:** если в предложении нет спрягаемого глагола — добавь. Выбери глагол, который говорит, что именно изменилось.

### Разбей и склей

Одно предложение — одна мысль. Но разбивка может дать разрозненные обрывки. Поэтому это цикл.

**Разбивка — примеры:**

| Было                                                                                                                                                                              | Стало                                                                                                               |
| --------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- | ------------------------------------------------------------------------------------------------------------------- |
| Мы предлагаем комплексную услугу по продвижению, включающую SEO, контент, рекламу и аналитику, что позволяет компании комплексно подходить к развитию, а также сокращать затраты. | Мы предлагаем: SEO, контент, рекламу, аналитику. Это позволяет подходить к развитию комплексно и сокращать затраты. |
| В связи с тем, что у нас появились новые условия сотрудничества, мы хотели бы обратить ваше внимание на некоторые важные моменты, которые, возможно, будут интересны вам.         | У нас новые условия сотрудничества. Вот главное.                                                                    |

**Склейка — примеры:**

| Было                                                                     | Стало                                                                       |
| ------------------------------------------------------------------------ | --------------------------------------------------------------------------- |
| HTML-файл с трекером активов. Двойная бухгалтерия, визуализация treemap. | HTML-файл с трекером активов, двойной бухгалтерией и визуализацией treemap. |
| Git хранит только структуру. Контент синхронизируется через livesync.    | Git хранит только структуру, а контент синхронизируется через livesync.     |
| Файлы копятся. Иногда перечитываю. Синхронизация через Syncthing.        | Файлы копятся и синхронизируются через Syncthing. Иногда перечитываю.       |
| Набор маленьких репозиториев. (повисло после предыдущего абзаца)         | В итоге получился набор маленьких репозиториев.                             |

**Способы склеить:**

- **Союз** — «и», «а», «но», «поэтому» объединяют два коротких предложения в одно
- **Общее подлежащее** — «она», «система», «этот файл» вместо нового подлежащего с нуля
- **Отсылка назад** — «в итоге», «для этого», «такой» привязывает к предыдущему предложению
- **Слияние в оборот** — сделай одно предложение частью другого («трекер с X и Y» вместо «трекер. X. Y.»)
- **Точка с запятой** (последний вариант) — только когда остальные техники не подходят и оба предложения короткие.

Источник: Williams, принцип 5 (cohesion)

### Прочитай вслух: запнулся — перепиши

Чтение вслух — лучший тест на читаемость. Запинаешься или теряешь дыхание — предложение перегружено.

| Было                                                                                            | Стало                                           |
| ----------------------------------------------------------------------------------------------- | ----------------------------------------------- |
| В течение долгого времени я, по правде сказать, не понимал, как поступить с моими сотрудниками. | Долго не мог решить, что делать с сотрудниками. |
| В ходе реализации проекта имело место быть отсутствие надлежащей коммуникации                   | Во время проекта команда почти не общалась      |

**Тест:** прочитай текст вслух. Где запнулся — там проблема.

### Ставь сказуемое ближе к началу

В длинных предложениях цепочки предлогов и обстоятельств отодвигают глагол. Читатель ждёт действия, а получает вступление.

| Было                                                                                                        | Стало                                                          |
| ----------------------------------------------------------------------------------------------------------- | -------------------------------------------------------------- |
| После долгих переговоров с представителями профсоюза по вопросам условий труда руководство приняло решение. | Руководство решило после переговоров с профсоюзом.             |
| В результате проведённого анализа данных за последний квартал текущего года команда выявила проблему.       | Команда проанализировала данные за квартал и выявила проблему. |

**Диагностика:** в предложениях длиннее 20 слов проверь: сказуемое во второй половине? Если вступление можно сократить или переставить — перестрой.

# Проход 3: Абзацы

## Псевдокод

```
for each абзац:
  if главная мысль спрятана (вывод после подводки):
    перенеси главное в первое предложение  // Начинай с главного
  if абзац раскрывает больше одной мысли:
    разбей на отдельные абзацы  // Один абзац — одна мысль

for each заголовок:
  if заголовок общий (Введение, Заключение, О компании):
    перепиши — скажи, о чём раздел  // Заголовок обещает конкретную пользу

for each раздел:
  if сплошной текст без подзаголовков и списков:
    добавь подзаголовки, списки, короткие абзацы  // Структурируй для сканирования
```

## Правила

### Начинай с главного

Первое предложение абзаца — самое важное. Читатель решает по нему, читать ли дальше.

| Было                                                                                                                              | Стало                                                                |
| --------------------------------------------------------------------------------------------------------------------------------- | -------------------------------------------------------------------- |
| Принимая во внимание все факторы, включая стоимость, сроки и риски, мы пришли к выводу, что стоит мигрировать на новую платформу. | Мигрируем на новую платформу. Стоимость, сроки и риски — приемлемые. |
| Существует множество факторов, которые влияют на производительность системы. Один из них — кэширование.                           | Кэширование ускоряет систему.                                        |

### Один абзац — одна мысль

Абзац на 15 предложений обо всём сразу теряет фокус. Один абзац раскрывает одну мысль: главное предложение, детали, примеры.

| Было                                                                       | Стало                                                  |
| -------------------------------------------------------------------------- | ------------------------------------------------------ |
| Абзац на 15 предложений обо всём сразу                                     | Три абзаца по 3–5 предложений, каждый об одном аспекте |
| Абзац про архитектуру, в середине — про тестирование, в конце — про деплой | Три абзаца: архитектура, тестирование, деплой          |

**Структура абзаца:** главное предложение → развитие мысли → (опционально) заверши деталью, которую следующий абзац подхватит, или начни следующий абзац с отсылки к предыдущему.

### Заголовок обещает конкретную пользу

Заголовок — обещание. Читатель по нему решает, читать ли раздел.

| Было        | Стало                                |
| ----------- | ------------------------------------ |
| Введение    | Зачем мигрировать на новую платформу |
| Наши услуги | Увеличим заявки на 30% за 3 месяца   |
| О компании  | 8 лет на рынке, 200 проектов         |
| Заключение  | Следующие шаги и сроки               |

**Тест:** прочитай только заголовки. Понятен ли текст без абзацев?

### Структурируй для сканирования

Большинство читателей сканирует текст, а не читает подряд. Помоги им: заголовки, подзаголовки, списки, короткие абзацы.

| Было                                | Стало                                                   |
| ----------------------------------- | ------------------------------------------------------- |
| Сплошной текст на 3 экрана          | Текст с подзаголовками через каждые 3–5 абзацев         |
| Перечисление через запятые в строку | Список с отдельными пунктами                            |
| Один блок обо всём                  | Один блок = одна крупная мысль, разделяй подзаголовками |

**Логика:** проблема → причины → способы решения → инструменты → выводы.

# Проход 4: AI-паттерны

У этих паттернов одна причина: модель тянет к среднему и сглаживает конкретику в общие, важно звучащие фразы. Одна проверка ловит большинство: подходит ли предложение почти к любому предмету? Если да — вырежи его или замени тем, что верно только для этого предмета.

## Псевдокод

```
for each предложение:
  if начинается с маркера («Важно отметить», «В современном мире», «Давайте разберёмся»):
    убери маркер, начни с сути  // Убирай фразы-маркеры AI-текста

for each прилагательное/модификатор:
  if рекламный штамп (широкий ассортимент, индивидуальный подход, инновационный):
    замени фактом, числом или названием  // Заменяй рекламные штампы конкретикой

for текст в целом:
  if все предложения одинаковой длины (однообразные 15–20 слов):
    чередуй: добавь короткие, дай некоторым вытянуться  // Чередуй короткие и длинные
  if связки избыточны («Кроме того», «Также», «В заключение»):
    убери, если логика и так понятна  // Ломай однообразную AI-структуру

for each пунктуация:
  if точка с запятой и хотя бы одно предложение длинное:
    разбей на два предложения  // Упрощай пунктуацию
  if скобки с важной информацией:
    вынеси в отдельное предложение  // Упрощай пунктуацию
  if больше трёх запятых:
    разбей на два предложения  // Упрощай пунктуацию

for each предложение:
  if раздувает значимость/наследие/тренд или кончается деепричастным оборотом-оценкой:
    вырежи или замени конкретным фактом с источником  // Убирай раздувание значимости
  if связку заменили на «является»/«представляет собой»/«выступает в качестве»:
    верни тире или прямой глагол  // Возвращай тире вместо «является»
  if «не просто X, а Y» / «это не X, это Y» / тройка-перечисление:
    скажи прямо; оставь в тройке только то, что верно  // Убирай шаблонные конструкции

for each слово:
  if из набора AI-лексики и он скапливается здесь:
    возьми простой глагол  // Прорежай AI-лексику
  if размытая атрибуция («эксперты считают», «ряд источников»):
    назови источник и точное утверждение или убери  // Прорежай AI-лексику
  if синоним подставлен только чтобы не повторяться:
    повтори простое слово  // Прорежай AI-лексику

guard:
  оставь простые формы, простые слова и настоящие превосходные степени; одиночный случай — не признак  // Сохраняй живое и понятное
```

## Правила

### Убирай фразы-маркеры AI-текста

Фразы-маркеры выдают машинный текст. Убирай их и начинай с сути.

| Было                                               | Стало                             |
| -------------------------------------------------- | --------------------------------- |
| Важно отметить, что система работает стабильно     | Система работает стабильно        |
| В современном мире технологии играют ключевую роль | Технологии влияют на X и Y        |
| Давайте разберёмся, как это работает               | Вот как это работает              |
| Таким образом, можно сделать вывод                 | (убрать, написать вывод напрямую) |

**Частые маркеры:** «важно отметить», «в современном мире», «на сегодняшний день», «является одним из», «не секрет, что», «играет ключевую роль», «следует учитывать».

### Заменяй рекламные штампы конкретикой

Рекламные штампы не несут информации. Замени каждый штамп фактом, числом или названием.

| Было                     | Стало                                                        |
| ------------------------ | ------------------------------------------------------------ |
| широкий ассортимент      | 200 товаров в 5 категориях                                   |
| индивидуальный подход    | назначаем менеджера, который ведёт проект от начала до конца |
| инновационные технологии | используем технологию X                                      |
| конкурентные цены        | от 500 ₽, дешевле рынка на 15%                               |

**10 частых штампов:** широкий ассортимент, индивидуальный подход, высокое качество, минимальный уход, конкурентные цены, инновационные технологии, быстрая доставка, широкий спектр, ряд преимуществ, идеальное решение.

### Ломай однообразную AI-структуру

AI-текст предсказуем: одинаковая длина предложений, каждый абзац с обобщения, избыточная «сбалансированность». Ломай шаблон.

| Было                                                                 | Стало                                                                      |
| -------------------------------------------------------------------- | -------------------------------------------------------------------------- |
| Каждый абзац начинается с обобщения, затем 3 пункта одинаковой длины | Начни один абзац с примера, другой — с вопроса, третий — с короткого факта |
| С одной стороны X, с другой стороны Y (когда позиция однозначна)     | X. Точка.                                                                  |
| Также следует отметить... Кроме того... В заключение...              | Убрать связки, если абзацы и так логически связаны                         |

**Приёмы:** чередуй длину предложений, вставь конкретный пример, убери «сбалансированные» оговорки, если позиция однозначна.

### Чередуй короткие и длинные предложения

Текст из одинаковых предложений усыпляет. Чередуй короткие и длинные — ритм удерживает внимание.

| Было                                                                                                           | Стало                                                                                     |
| -------------------------------------------------------------------------------------------------------------- | ----------------------------------------------------------------------------------------- |
| Система обрабатывает запросы. Система проверяет данные. Система возвращает результат. Система логирует ошибки. | Система обрабатывает запросы и проверяет данные. Ошибки логируются. Результат — за 50 мс. |

**Приём:** после длинного объяснения — короткое предложение. Это работает.

### Упрощай пунктуацию

Точка — лучший знак препинания. Если сомневаешься в пунктуации — упрости предложение.

| Было                                                                                                                | Стало                                                                                   |
| ------------------------------------------------------------------------------------------------------------------- | --------------------------------------------------------------------------------------- |
| Система, которая была разработана нашей командой, и которая, по мнению экспертов, является лучшей, запущена в прод. | Наша команда разработала систему. Эксперты считают её лучшей на рынке. Система в проде. |
| Результат (впрочем, не окончательный) оказался неожиданным.                                                         | Предварительный результат оказался неожиданным.                                         |
| Мы хотели бы отметить... что проект движется вперёд.                                                                | Проект движется вперёд.                                                                 |

**Заменяй тире другой пунктуацией при склейке предложений.** AI-текст злоупотребляет тире как универсальным соединителем. Вместо тире: раздели на два предложения, используй союз, двоеточие или запятую. Тире допустимо только в устойчивых конструкциях («один абзац — одна мысль») и при обособлении приложений.

**Дефис (-) и тире (—):** дефис только внутри слов (какой-то, онлайн-курс), без пробелов. Тире длинное, с пробелами. В русской типографике среднего тире нет.

**Больше трёх запятых** в предложении: разбей на два.

**Точка с запятой** допустима, когда оба предложения короткие (см. проход 2). Если хотя бы одно длинное — разбей точкой.

**Скобки:** информация в них выглядит второстепенной. Убери или вынеси в отдельное предложение. Оставляй только для ссылок и источников.

**Многоточие:** только для пропусков в цитатах. Для неуверенности и пауз не годится.

### Убирай раздувание значимости

Фразы про важность, наследие или «более широкие процессы» ничего не говорят о предмете. Убери их или замени конкретным фактом с источником. Деепричастный оборот-оценка в конце предложения — то же самое на уровне оборота.

| Было                                                                                            | Стало                         |
| ----------------------------------------------------------------------------------------------- | ----------------------------- |
| Институт основан в 1989 году, что стало поворотным моментом в развитии региональной статистики. | Институт основан в 1989 году. |
| На станции восемь путей, что вносит вклад в социально-экономическое развитие региона.           | На станции восемь путей.      |
| Эта этимология подчёркивает непреходящее наследие сопротивления общины.                         | (убрать)                      |

**Маркеры:** играет ключевую роль, знаменует поворотный момент, оставил неизгладимый след, отражает более широкие процессы, подчёркивает значимость, открывает новую главу, в эпоху.

### Возвращай тире вместо «является»

В русском настоящем времени связки нет: между подлежащим и сказуемым ставят тире. AI подставляет «является», «представляет собой», «выступает в качестве». Верни тире или прямой глагол.

| Было                                                     | Стало                                          |
| -------------------------------------------------------- | ---------------------------------------------- |
| Галерея является выставочным пространством ассоциации.   | Галерея — выставочное пространство ассоциации. |
| Платформа представляет собой одно из крупнейших решений. | Платформа — одно из крупнейших решений.        |
| Компания выступает в качестве поставщика оборудования.   | Компания поставляет оборудование.              |

**Замены:** является / представляет собой / выступает в качестве / служит → тире или прямой глагол.

### Убирай шаблонные конструкции

Две привычные машинные конструкции. «Не просто X, а Y», «это не X, это Y» — выдуманное возражение, которое тут же опровергают. Тройка-перечисление добавляет к одному смыслу два почти-синонима. Скажи прямо; оставь только то, что верно.

| Было                                                         | Стало                                 |
| ------------------------------------------------------------ | ------------------------------------- |
| Портрет — не просто автопортрет, а документ её одержимостей. | Портрет документирует её одержимости. |
| Ясное, краткое и убедительное резюме.                        | Ясное резюме.                         |
| Платформа быстрая, надёжная и масштабируемая.                | Платформа держит 50 000 соединений.   |

### Прорежай AI-лексику

Набор повторяющихся слов выдаёт машинный текст плотностью, а не одним употреблением. Где набор скапливается, бери простой глагол. Рядом идут два спутника.

**Набор:** подчёркивает, отражает, играет ключевую роль, в условиях, на фоне, ландшафт, палитра, неотъемлемая часть, богатое наследие, динамично развивающийся, призван, способствует, целый ряд, уникальный, комплексный.

- **Размытая атрибуция:** «эксперты считают», «по мнению наблюдателей», «ряд источников» (когда источник один). Назови источник и точное утверждение или убери.
- **Подмена синонимов:** «компания… фирма… организация». Повтори простое слово («компания… она… компания»). Перебор синонимов — машинный приём, человек повторяет.

### Сохраняй живое и понятное

Признак — плотность и совпадение, а не один случай. Оставь живое и понятное:

- Простые формы: «это», прямую форму без глагола, тире.
- Простые слова вместо вычурных: написал (не «осуществил написание»), использовал (не «задействовал»), умер (не «ушёл из жизни»).
- Настоящие превосходные степени, когда это факт: первый, единственный, крупнейший.
- Одно тире, одно «Кроме того» — не признак AI. Останови правку, пока текст остаётся живым.
