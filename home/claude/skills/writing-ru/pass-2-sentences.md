# Проход 2: Предложения

## Псевдокод

```
for each предложение:
  if глагол в пассивном залоге и деятель известен:
    перепиши в активном залоге  // Используй активный залог
  if причастный/деепричастный оборот длиннее 4 слов:
    разверни в придаточное с «который» или отдельное предложение  // Разворачивай причастные обороты
  if нет сказуемого и смысл держится на контрасте («но», «а не», «зато»):
    перепиши с подлежащим и сказуемым  // Разворачивай безглагольные обрывки

// Цикл разбивки и склейки
loop:
  for each предложение:
    while больше одной мысли:
      разбей → цель 15–25 слов на кусок  // Разбей и склей
  for each соседняя пара:
    if об одном, но звучат как чужие:
      склей: союз / общее подлежащее / отсылка назад / слияние в оборот / тире
      if склеенное перегружено:
        перефразируй другой техникой
        if после 2 попыток всё ещё перегружено:
          покажи оба варианта пользователю → stop
      go to loop

// Финальная проверка
for each предложение:
  if запинаешься при чтении вслух:
    перепиши  // Прочитай вслух
```

## Правила

### Используй активный залог

Активный залог показывает, кто действует. Пассивный прячет деятеля и усложняет понимание.

| Было                           | Стало                      |
| ------------------------------ | -------------------------- |
| Запрос обрабатывается сервером | Сервер обрабатывает запрос |
| Решение было принято           | Команда приняла решение    |
| Ошибка была обнаружена         | Тест обнаружил ошибку      |
| Данные были загружены          | Система загрузила данные   |

**Исключение:** пассив допустим, когда деятель неизвестен или неважен: «Письмо отправлено.»

### Разворачивай причастные обороты

Причастные и деепричастные обороты перегружают предложение. Разверни в придаточное с «который» или отдельное предложение.

| Было                                               | Стало                                                   |
| -------------------------------------------------- | ------------------------------------------------------- |
| Пользователь, воспользовавшийся сервисом           | Пользователь, который воспользовался сервисом           |
| Он был прав, спрашивая вас                         | Он правильно сделал, что спросил вас                    |
| Обеспечивая надёжность, система работает стабильно | Система работает стабильно: она обеспечивает надёжность |

**Тест:** если причастный оборот длиннее четырёх слов — разворачивай.

### Разворачивай безглагольные обрывки

Фрагменты без сказуемого, которые держатся на контрасте («но», «а не», «зато»). Читатель сам достраивает пропущенный глагол. Скажи прямо, что произошло.

| Было                                              | Стало                                |
| ------------------------------------------------- | ------------------------------------ |
| Реальный прогресс, но в окружении, а не в модели. | Улучшилось только окружение.         |
| Хорошие результаты, но только локально.           | Результаты улучшились локально.      |
| Больше фич, меньше стабильности.                  | Добавление фич снизило стабильность. |
| Быстро, но хрупко.                                | Работает быстро, но легко ломается.  |

**Тест:** если в предложении нет спрягаемого глагола — добавь. Выбери глагол, который говорит, что именно изменилось.

### Разбей и склей

Одно предложение — одна мысль. Но разбивка может дать разрозненные обрывки. Поэтому это цикл.

**Разбивка — примеры:**

| Было                                                                                                                                                                              | Стало                                                                                                               |
| --------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- | ------------------------------------------------------------------------------------------------------------------- |
| Мы предлагаем комплексную услугу по продвижению, включающую SEO, контент, рекламу и аналитику, что позволяет компании комплексно подходить к развитию, а также сокращать затраты. | Мы предлагаем: SEO, контент, рекламу, аналитику. Это позволяет подходить к развитию комплексно и сокращать затраты. |
| В связи с тем, что у нас появились новые условия сотрудничества, мы хотели бы обратить ваше внимание на некоторые важные моменты, которые, возможно, будут интересны вам.         | У нас новые условия сотрудничества. Вот главное.                                                                    |

**Склейка — примеры:**

| Было                                                                     | Стало                                                                       |
| ------------------------------------------------------------------------ | --------------------------------------------------------------------------- |
| HTML-файл с трекером активов. Двойная бухгалтерия, визуализация treemap. | HTML-файл с трекером активов, двойной бухгалтерией и визуализацией treemap. |
| Git хранит только структуру. Контент синхронизируется через livesync.    | Git хранит только структуру, а контент синхронизируется через livesync.     |
| Файлы копятся. Иногда перечитываю. Синхронизация через Syncthing.        | Файлы копятся и синхронизируются через Syncthing. Иногда перечитываю.       |
| Набор маленьких репозиториев. (повисло после предыдущего абзаца)         | В итоге получился набор маленьких репозиториев.                             |

**Способы склеить:**

- **Союз** — «и», «а», «но», «поэтому» объединяют два коротких предложения в одно
- **Общее подлежащее** — «она», «система», «этот файл» вместо нового подлежащего с нуля
- **Отсылка назад** — «в итоге», «для этого», «такой» привязывает к предыдущему предложению
- **Слияние в оборот** — сделай одно предложение частью другого («трекер с X и Y» вместо «трекер. X. Y.»)
- **Точка с запятой** (последний вариант) — только когда остальные техники не подходят и оба предложения короткие.

Источник: Williams, принцип 5 (cohesion)

### Прочитай вслух: запнулся — перепиши

Чтение вслух — лучший тест на читаемость. Запинаешься или теряешь дыхание — предложение перегружено.

| Было                                                                                            | Стало                                           |
| ----------------------------------------------------------------------------------------------- | ----------------------------------------------- |
| В течение долгого времени я, по правде сказать, не понимал, как поступить с моими сотрудниками. | Долго не мог решить, что делать с сотрудниками. |
| В ходе реализации проекта имело место быть отсутствие надлежащей коммуникации                   | Во время проекта команда почти не общалась      |

**Тест:** прочитай текст вслух. Где запнулся — там проблема.
