# Ясное письмо — полный справочник

Источники: Нора Галь («Слово живое и мёртвое»), Ильяхов/Сарычева («Пиши, сокращай»), 1ps.ru.

## Действие

### Ставь глагол в центр предложения

Глагол — двигатель предложения. Отглагольные существительные (-ание, -ение, -ция) утяжеляют текст и прячут действие. Верни действие в глагол.

| Было                             | Стало                    |
| -------------------------------- | ------------------------ |
| осуществление поддержки клиентов | поддерживаем клиентов    |
| произвести установку             | установить               |
| выполнить оптимизацию            | оптимизировать           |
| обеспечить функционирование      | запустить / поддерживать |

**Тест:** если можно убрать «произвести/осуществить/выполнить» и поставить глагол — убирай.

### Используй активный залог

Активный залог показывает, кто действует. Пассивный залог прячет деятеля и усложняет понимание. Найди актора и сделай его подлежащим.

| Было                           | Стало                      |
| ------------------------------ | -------------------------- |
| Запрос обрабатывается сервером | Сервер обрабатывает запрос |
| Решение было принято           | Команда приняла решение    |
| Ошибка была обнаружена         | Тест обнаружил ошибку      |
| Данные были загружены          | Система загрузила данные   |

**Исключение:** пассив допустим, когда деятель неизвестен или неважен: «Письмо отправлено.»

### Разворачивай причастные обороты

Причастные и деепричастные обороты перегружают предложение. Разверни их в отдельное предложение или придаточное с «который».

| Было                                               | Стало                                                   |
| -------------------------------------------------- | ------------------------------------------------------- |
| Пользователь, воспользовавшийся сервисом           | Пользователь, который воспользовался сервисом           |
| Он был прав, спрашивая вас                         | Он правильно сделал, что спросил вас                    |
| Обеспечивая надёжность, система работает стабильно | Система работает стабильно: она обеспечивает надёжность |

**Тест:** если причастный оборот длиннее четырёх слов — разворачивай.

## Выбор слов

### Выбирай короткое слово

Короткое слово понятнее длинного. Если есть простой синоним — бери его.

| Было                      | Стало    |
| ------------------------- | -------- |
| на сегодняшний день       | сейчас   |
| осуществлять деятельность | работать |
| произвести оплату         | оплатить |
| в целях обеспечения       | чтобы    |

### Убирай слова без смысловой нагрузки

Каждое слово должно нести смысл. Если убрать слово и смысл предложения сохранился — слово было мусором.

| Было                                        | Стало               |
| ------------------------------------------- | ------------------- |
| Следует отметить, что система работает      | Система работает    |
| Ни для кого не секрет, что цены растут      | Цены растут         |
| В принципе, это работает достаточно хорошо  | Это работает хорошо |
| Как уже было сказано выше, данные обновлены | Данные обновлены    |

**Слова-паразиты:** «в принципе», «в целом», «как бы», «собственно говоря», «достаточно», «довольно», «весьма».

**Тест:** убери слово. Смысл сохранился? Значит, слово было мусором.

### Пиши утверждениями

Отрицания требуют дополнительного усилия: читатель сначала представляет действие, потом отменяет его. Утверждение сразу даёт правильную картину.

| Было                    | Стало          |
| ----------------------- | -------------- |
| Не забудьте сохранить   | Сохраните      |
| Он не помнил            | Он забыл       |
| Нельзя не отметить      | Стоит отметить |
| Не является невозможным | Возможно       |

**Двойное отрицание — враг ясности.** «Не следует игнорировать» → «Учитывайте».

### Пиши живым языком

Канцелярский язык отталкивает читателя. Пиши так, как говорят живые люди (в рамках литературной нормы).

| Было                     | Стало             |
| ------------------------ | ----------------- |
| в рамках данного проекта | в этом проекте    |
| ввиду отсутствия         | так как нет       |
| имеет место быть         | есть / происходит |
| вышеуказанный            | этот              |
| в связи с тем, что       | так как           |
| при условии, что         | если              |

## Предложение

### Одно предложение — одна мысль

Длинное предложение с двумя и более запятыми — кандидат на разбивку. Одна мысль в одном предложении легче читается и запоминается.

| Было                                                                                                                                                                              | Стало                                                                                                               |
| --------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- | ------------------------------------------------------------------------------------------------------------------- |
| Мы предлагаем комплексную услугу по продвижению, включающую SEO, контент, рекламу и аналитику, что позволяет компании комплексно подходить к развитию, а также сокращать затраты. | Мы предлагаем: SEO, контент, рекламу, аналитику. Это позволяет подходить к развитию комплексно и сокращать затраты. |
| В связи с тем, что у нас появились новые условия сотрудничества, мы хотели бы обратить ваше внимание на некоторые важные моменты, которые, возможно, будут интересны вам.         | У нас новые условия сотрудничества. Вот главное.                                                                    |

### Чередуй короткие и длинные предложения

Текст из одинаковых предложений усыпляет. Чередование коротких и длинных создаёт ритм, который удерживает внимание.

| Было                                                                                                           | Стало                                                                                     |
| -------------------------------------------------------------------------------------------------------------- | ----------------------------------------------------------------------------------------- |
| Система обрабатывает запросы. Система проверяет данные. Система возвращает результат. Система логирует ошибки. | Система обрабатывает запросы и проверяет данные. Ошибки логируются. Результат — за 50 мс. |

**Приём:** после длинного объяснения — короткое предложение. Это работает.

### Прочитай вслух: запнулся — перепиши

Чтение вслух — лучший тест на читаемость. Если запинаешься, спотыкаешься или теряешь дыхание — предложение перегружено.

| Было                                                                                            | Стало                                           |
| ----------------------------------------------------------------------------------------------- | ----------------------------------------------- |
| В течение долгого времени я, по правде сказать, не понимал, как поступить с моими сотрудниками. | Долго не мог решить, что делать с сотрудниками. |
| В ходе реализации проекта имело место быть отсутствие надлежащей коммуникации                   | Во время проекта команда почти не общалась      |

**Тест:** прочитай текст вслух. Где запнулся — там проблема.

## Абзац и структура

### Начинай с главного

Первое предложение абзаца — самое важное. Читатель решает по нему, стоит ли читать дальше. Ставь главную мысль первой, детали — потом.

| Было                                                                                                                              | Стало                                                                |
| --------------------------------------------------------------------------------------------------------------------------------- | -------------------------------------------------------------------- |
| Принимая во внимание все факторы, включая стоимость, сроки и риски, мы пришли к выводу, что стоит мигрировать на новую платформу. | Мигрируем на новую платформу. Стоимость, сроки и риски — приемлемые. |
| Существует множество факторов, которые влияют на производительность системы. Один из них — кэширование.                           | Кэширование ускоряет систему.                                        |

### Один абзац — одна мысль

Абзац на 15 предложений обо всём сразу — нечитаем. Один абзац раскрывает одну мысль: главное предложение, детали, примеры.

| Было                                                                       | Стало                                                  |
| -------------------------------------------------------------------------- | ------------------------------------------------------ |
| Абзац на 15 предложений обо всём сразу                                     | Три абзаца по 3–5 предложений, каждый об одном аспекте |
| Абзац про архитектуру, в середине — про тестирование, в конце — про деплой | Три абзаца: архитектура, тестирование, деплой          |

**Структура абзаца:** главное предложение → развитие мысли → связь со следующим абзацем (опционально).

### Заголовок обещает конкретную пользу

Заголовок — обещание. Читатель по заголовку решает, читать ли раздел. Конкретный заголовок работает лучше абстрактного.

| Было        | Стало                                |
| ----------- | ------------------------------------ |
| Введение    | Зачем мигрировать на новую платформу |
| Наши услуги | Увеличим заявки на 30% за 3 месяца   |
| О компании  | 8 лет на рынке, 200 проектов         |
| Заключение  | Следующие шаги и сроки               |

**Тест:** прочитай только заголовки. Понятен ли текст без абзацев?

### Структурируй для сканирования

Большинство читателей сканирует текст, а не читает подряд. Помоги им: заголовки, подзаголовки, списки, короткие абзацы.

| Было                                | Стало                                                   |
| ----------------------------------- | ------------------------------------------------------- |
| Сплошной текст на 3 экрана          | Текст с подзаголовками через каждые 3–5 абзацев         |
| Перечисление через запятые в строку | Список с отдельными пунктами                            |
| Один блок обо всём                  | Один блок = одна крупная мысль, разделяй подзаголовками |

**Логика:** проблема → причины → способы решения → инструменты → выводы.

## Читатель

### Пиши для конкретного читателя

Говори о проблемах читателя, а не о себе. Используй примеры из жизни читателя. Объясняй термины с учётом его уровня.

| Было                                                        | Стало                                |
| ----------------------------------------------------------- | ------------------------------------ |
| Компания «Вася и Ко» отмечает юбилей и приглашает партнеров | Мы отмечаем юбилей и приглашаем вас  |
| Наша команда профессионалов мирового уровня                 | У специалистов в среднем 8 лет опыта |
| Мы предлагаем уникальный продукт                            | Продукт решает проблему X за Y минут |

**Принцип Ильяхова:** сними корону с себя, надень на читателя.

### Заменяй абстракции фактами и числами

Абстрактные слова ничего не сообщают. Если можно заменить слово числом, датой или фактом — замени.

| Было                                    | Стало                   |
| --------------------------------------- | ----------------------- |
| работает быстро                         | отвечает за 50 мс       |
| много пользователей                     | 12 000 пользователей    |
| значительно улучшили                    | ускорили в 3 раза       |
| недавно обновили                        | обновили 15 января      |
| квартира в шаговой доступности от метро | 5 минут пешком до метро |

### Говори об ограничениях

Говори о том, что продукт делает плохо или кому он не подойдёт. Честность выигрывает доверие читателя.

| Было                         | Стало                                                                          |
| ---------------------------- | ------------------------------------------------------------------------------ |
| Идеальное решение для всех   | Подходит для команд от 5 до 50 человек. Для больших команд — другой инструмент |
| Самый лучший сервис на рынке | Быстрый сервис для типовых задач. Сложные кейсы требуют настройки              |

**Принцип:** быть честным — значит говорить правду, даже неудобную.

## AI-паттерны

### Убирай фразы-маркеры AI-текста

Фразы-маркеры выдают машинный текст. Убирай их и начинай с сути.

| Было                                               | Стало                             |
| -------------------------------------------------- | --------------------------------- |
| Важно отметить, что система работает стабильно     | Система работает стабильно        |
| В современном мире технологии играют ключевую роль | Технологии влияют на X и Y        |
| Давайте разберёмся, как это работает               | Вот как это работает              |
| Таким образом, можно сделать вывод                 | (убрать, написать вывод напрямую) |

**Частые маркеры:** «важно отметить», «в современном мире», «на сегодняшний день», «является одним из», «не секрет, что», «играет ключевую роль», «следует учитывать».

### Заменяй рекламные штампы конкретикой

Рекламные штампы не несут информации. Замени каждый штамп фактом, числом или названием.

| Было                     | Стало                                                        |
| ------------------------ | ------------------------------------------------------------ |
| широкий ассортимент      | 200 товаров в 5 категориях                                   |
| индивидуальный подход    | назначаем менеджера, который ведёт проект от начала до конца |
| инновационные технологии | используем технологию X                                      |
| конкурентные цены        | от 500 ₽, дешевле рынка на 15%                               |

**10 частых штампов:** широкий ассортимент, индивидуальный подход, высокое качество, минимальный уход, конкурентные цены, инновационные технологии, быстрая доставка, широкий спектр, ряд преимуществ, идеальное решение.

### Ломай однообразную AI-структуру

AI-текст предсказуем: одинаковая длина предложений, каждый абзац с обобщения, избыточная «сбалансированность». Ломай шаблон.

| Было                                                                 | Стало                                                                      |
| -------------------------------------------------------------------- | -------------------------------------------------------------------------- |
| Каждый абзац начинается с обобщения, затем 3 пункта одинаковой длины | Начни один абзац с примера, другой — с вопроса, третий — с короткого факта |
| С одной стороны X, с другой стороны Y (когда позиция однозначна)     | X. Точка.                                                                  |
| Также следует отметить... Кроме того... В заключение...              | Убрать связки, если абзацы и так логически связаны                         |

**Приёмы:** чередуй длину предложений, вставь конкретный пример, убери «сбалансированные» оговорки, если позиция однозначна.

### Упрощай пунктуацию

Точка — лучший знак препинания. Если сомневаешься в пунктуации — упрости предложение.

| Было                                                                                                                | Стало                                                                                   |
| ------------------------------------------------------------------------------------------------------------------- | --------------------------------------------------------------------------------------- |
| Система, которая была разработана нашей командой, и которая, по мнению экспертов, является лучшей, запущена в прод. | Наша команда разработала систему. Эксперты считают её лучшей на рынке. Система в проде. |
| Результат (впрочем, не окончательный) оказался неожиданным.                                                         | Предварительный результат оказался неожиданным.                                         |
| Мы хотели бы отметить... что проект движется вперёд.                                                                | Проект движется вперёд.                                                                 |

**Тире (—)** — длинное, с пробелами по обе стороны. Дефис (-) — только внутри слов (какой-то, онлайн-курс), без пробелов. В русской типографике среднего тире нет.

**Больше трёх запятых** в предложении — разбей на два.

**Точка с запятой** — предложение слишком сложное. Разбей.

**Скобки** — информация в них кажется неважной. Убери или вынеси в отдельное предложение. Оставляй только для ссылок и источников.

**Многоточие** — только для пропусков в цитатах. Для неуверенности и пауз — дилетантство.
