# Ясное письмо на русском

Источники: Нора Галь («Слово живое и мёртвое»), Ильяхов/Сарычева («Пиши, сокращай»), 1ps.ru.

## Проходы

Запускай каждый проход отдельным субагентом (чистый контекст на проход), строго по одному. Каждый проход правит результат предыдущего, поэтому дождись завершения прохода перед запуском следующего. Каждый файл содержит псевдокод сверху и полные правила ниже.

| Проход | Файл                                           | Что правит                                                                    |
| ------ | ---------------------------------------------- | ----------------------------------------------------------------------------- |
| 1      | [pass-1-words.md](pass-1-words.md)             | Слова: отглагольные, длинные слова, мусор, отрицания, канцелярит              |
| 2      | [pass-2-sentences.md](pass-2-sentences.md)     | Предложения: активный залог, причастные обороты, разбей+склей, чтение вслух   |
| 3      | [pass-3-paragraphs.md](pass-3-paragraphs.md)   | Абзацы: главное первым, один абзац — одна мысль, заголовки, сканирование      |
| 4      | [pass-4-ai.md](pass-4-ai.md)                   | AI-паттерны: маркеры, штампы, однообразная структура, ритм, пунктуация        |

Применяй только те правила, которые улучшают текст.
