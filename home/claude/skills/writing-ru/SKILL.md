# Ясное письмо на русском

Источники: Нора Галь («Слово живое и мёртвое»), Ильяхов/Сарычева («Пиши, сокращай»), 1ps.ru.

## Правила

Применяй все правила за один проход. Детали правил в файлах ниже. Применяй только те правила, которые улучшают текст.

| Область      | Файл                                           | Что правит                                                                    |
| ------------ | ---------------------------------------------- | ----------------------------------------------------------------------------- |
| Слова        | [pass-1-words.md](pass-1-words.md)             | Отглагольные, длинные слова, мусор, отрицания, канцелярит                     |
| Предложения  | [pass-2-sentences.md](pass-2-sentences.md)     | Активный залог, причастные обороты, разбей+склей, чтение вслух                |
| Абзацы       | [pass-3-paragraphs.md](pass-3-paragraphs.md)   | Главное первым, один абзац — одна мысль, заголовки, сканирование              |
| AI-паттерны  | [pass-4-ai.md](pass-4-ai.md)                   | Маркеры, штампы, однообразная структура, ритм, пунктуация                     |
