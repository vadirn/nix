# Ясное письмо на русском

Источники: Нора Галь («Слово живое и мёртвое»), Ильяхов/Сарычева («Пиши, сокращай»), Williams (Style: Lessons in Clarity and Grace), 1ps.ru.

## Правила

Применяй все правила за один проход. Детали правил в файлах ниже. Пропускай правило только если оно изменит смысл, снизит точность или нарушит регистр текста (например, раскрывать тезисные списки в связную прозу).

| Область     | Файл                                         | Что правит                                                                           |
| ----------- | -------------------------------------------- | ------------------------------------------------------------------------------------ |
| Слова       | [pass-1-words.md](pass-1-words.md)           | Отглагольные, длинные слова, мусор, отрицания, мёртвые метафоры, канцелярит          |
| Предложения | [pass-2-sentences.md](pass-2-sentences.md)   | Связность (known-new), единая тема, активный залог, разбей+склей, позиция сказуемого |
| Абзацы      | [pass-3-paragraphs.md](pass-3-paragraphs.md) | Главное первым, один абзац — одна мысль, заголовки, сканирование                     |
| AI-паттерны | [pass-4-ai.md](pass-4-ai.md)                 | Маркеры, штампы, однообразная структура, ритм, пунктуация                            |
